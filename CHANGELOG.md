# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.1.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [1.0.0] - 2026-06-16

### Added
- Initial release.
- GitHub API integration to read public profile, repositories, languages, and stars.
- Full editing form pre-filled from real GitHub data.
- 15 section types: about/intro, skill icons, tech-stack badges, stats cards, activity graph, project showcase, trophies, social links, dev quote, support/sponsor, profile views + snake, Mermaid mindmap, collapsible journey timeline, FAQ, and footer.
- Deep styling: 6 themes, 8 accent colors + custom picker, 5 header styles, 4 badge styles, horizontal/vertical badge layout, 5 stats-card layouts.
- Live GitHub-faithful preview with light/dark toggle.
- Toggle and drag-to-reorder sections.
- Repo picker for project showcase.
- Copy markdown and download `README.md`.
